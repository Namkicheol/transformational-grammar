C-command & Anaphor Binding

§3.3 — C-command Condition on Anaphors · 문법성 판단 및 c-command 분석
Q01O / X
다음 문장의 문법성을 판단하시오.
The soldiers might disgrace themselves.
정답: ⭕ 정문
[the soldiers]의 first branching node는 S이고, S는 [themselves]를 dominate합니다.
→ [the soldiers] c-commands [themselves] ✓
→ C-command Condition on Anaphors 충족 → 정문
Q02O / X
다음 문장의 문법성을 판단하시오.
The soldiers' behaviour might disgrace themselves.
정답: ❌ 비문
[the soldiers']의 first branching node는 NP ([the soldiers' behaviour])입니다.
NP는 [themselves]를 dominate하지 않으므로,
→ [the soldiers'] c-commands [themselves] ✗ (precede ✓ / c-command ✗)
→ C-command Condition on Anaphors 불충족 → 비문
Q03O / X
다음 문장의 문법성을 판단하시오.
John will buy himself another car.
정답: ⭕ 정문
John의 first branching node는 S이고, S는 [himself]를 dominate합니다.
→ John c-commands himself ✓
→ antecedent John, anaphor himself → 인칭·수·성 일치 ✓ → 정문
Q04O / X
다음 문장의 문법성을 판단하시오.
John considers himself to be the champion.
정답: ⭕ 정문
consider는 Subject-to-Object Raising(SOR)을 허용하는 동사입니다.
himself는 [himself to be the champion] 절의 주어였지만, SOR로 인해 절 boundary 밖으로 이동합니다.
새로운 binding domain 안에서 John이 himself를 c-command ✓
→ C-command Condition on Anaphors 충족 → 정문
Q05객관식
다음 중 비문인 것을 모두 고른 것은?
ㄱ. Each other are always arguing.
ㄴ. John and Mary despise each other.
ㄷ. Themselves are very happy.
ㄹ. John and Mary despise themselves.
정답: ③ ㄱ, ㄷ
ㄱ. *Each other are always arguing. — antecedent 없음 → 비문
ㄷ. *Themselves are very happy. — antecedent 없음 → 비문

ㄴ. John and Mary despise each other. — [John and Mary]가 each other를 c-command ✓ → 정문
ㄹ. John and Mary despise themselves. — [John and Mary]가 themselves를 c-command ✓ → 정문
Q06객관식
*John despises each other.가 비문인 이유로 가장 적절한 것은?
정답: ②
each other는 두 개 이상의 개체를 지칭하는 antecedent가 필요한 reciprocal anaphor입니다.
John은 단수 → each other의 antecedent 조건 불충족
※ c-command 자체는 John → each other ✓이지만, appropriateness 조건 불충족으로 비문.
Q07빈칸
다음 빈칸에 영어로 쓰시오.

The men will shoot the arrows at each other.에서 each other의 antecedent가 two ways로 해석되는 이유는, the men과 the arrows 둘 다 each other를 ________하기 때문이다.
정답: c-command
The men will shoot the arrows at each other.
→ the men: first branching node = S → S dominates each other ✓ → c-command ✓
→ the arrows: first branching node = VP → VP dominates each other ✓ → c-command ✓
두 NP 모두 c-command 조건을 만족 → 구조적 중의성(structural ambiguity)

NPI ever & C-command

§3.3 — Negative Polarity Item 'ever'의 문법성 조건
Q08O / X
다음 문장의 문법성을 판단하시오.
Nobody will ever forgive you for that.
정답: ⭕ 정문
Nobody는 negative determiner 'no'를 immediate constituent로 포함하는 NP입니다.
nobody → ever를 precede ✓, c-command ✓
→ NPI 'ever'의 두 조건 모두 충족 → 정문
Q09O / X
다음 문장의 문법성을 판단하시오.
I will ever forgive you for that.
정답: ❌ 비문
ever는 영어의 긍정문(positive sentence)에서는 일반적으로 사용될 수 없는 NPI(Negative Polarity Item)입니다.
부정 요소(negative modal / negative Det 'no' 포함 NP)가 없으므로 → 비문
Q10O / X
다음 문장의 문법성을 판단하시오.
No Frenchman ever would help an Englishman.
정답: ⭕ 정문
No Frenchman은 negative Det 'no'를 immediate constituent로 포함하는 NP입니다.
No Frenchman → ever를 precede ✓, c-command ✓
→ NPI 조건 충족 → 정문
Q11객관식
다음 중 문법적인 문장을 모두 고른 것은?
ㄱ. No Frenchman would ever help an Englishman.
ㄴ. *Some Frenchmen ever wouldn't help an Englishman.
ㄷ. No Frenchman would help an Englishman ever.
ㄹ. *Some Frenchmen who don't like English people would ever help an Englishman.
정답: ② ㄱ, ㄷ
ㄱ. ✓ No Frenchman → ever precede ✓, c-command ✓
ㄷ. ✓ No Frenchman → ever(문장 끝) precede ✓, c-command ✓

ㄴ. ✗ Some Frenchmen ever wouldn't — negative item(wouldn't)이 ever 뒤에 위치 → precede ✗
ㄹ. ✗ don't은 관계절 [who don't like English people] 내부 → ever를 c-command ✗
Q12객관식
다음 세 문장이 모두 비문인 이유로 가장 적절한 것은?
(a) *Some Frenchmen ever would help no Englishman.
(b) *Some Frenchmen would ever help no Englishman.
(c) *Some Frenchmen would help no Englishman ever.
정답: ③
(a)(b): no Englishman이 ever 뒤에 위치 → precede ✗ → 비문
(c): no Englishman이 ever 앞에 있지만(precede ✓), VP 내부에 있어 ever(S의 immediate constituent)를 c-command ✗ → 비문

NPI 조건: ever는 negative constituent에 의해 preceded AND c-commanded 되어야 함
Q13빈칸
다음 빈칸에 영어로 쓰시오.

*I will ever forgive nobody for that.이 비문인 이유:
nobody는 ever보다 뒤에 위치하므로 ever를 ________ 하지 못하기 때문이다.
정답: precede
*I will ever forgive nobody for that.
nobody는 ever보다 뒤에 위치 → precede ✗
(c-command도 ✗이지만, 더 근본적으로 precede 조건부터 실패)
cf. I will ever forgive nobody → nobody가 ever 뒤 → precede ✗ → 비문

Conflating Categories & Category Tests

§3.6–3.8 — Particle/Preposition 구분, Adjective/Determiner 판별, cleft 테스트
Q14O / X
다음 문장의 문법성을 판단하시오.
He waited until.
정답: ❌ 비문
until은 NP complement 또는 Clause complement를 반드시 필요로 합니다.
cf. He waited until her departure. ✓ (NP complement)
cf. He waited until she left. ✓ (Clause complement)
complement 없이 단독 사용 불가 → Preposition 분석의 예외 사례 (subcategorization restriction)
Q15O / X
다음 문장의 문법성을 판단하시오.
He left during the party was still going on.
정답: ❌ 비문
during은 NP complement만 취할 수 있고, Clause complement는 취할 수 없습니다.
cf. He left during the party. ✓ (NP complement)
cf. *He left during. ✗ (complement 없음)
→ during은 Preposition이지만 NP complement만 허용 → Clause는 비문
Q16객관식
밑줄 친 단어의 범주(category)를 바르게 짝지은 것은?
(a) He's been difficult ever since the party.
(b) He's been difficult ever since the party began.
(c) He's been difficult ever since.
정답: ②
(a) Preposition — NP complement [the party] 동반
(b) Conjunction — Clause complement [the party began] 동반
(c) Particle — complement 없이 단독 사용

세 가지 용법 모두 가능한 since/after/before 등은 Preposition으로 통합 분석합니다.
Q17객관식
다음 중 Cleft sentence의 focus position에 올 수 없는 것은?
(a) It was a car that she bought.
(b) It was in the shop that I met her.
(c) *It is very pretty that she is.
(d) *It is go home that I will.
정답: ③ NP와 PP만 가능
Cleft sentence의 focus position에는 NP와 PP만 올 수 있습니다.
(a) a car = NP ✓
(b) in the shop = PP ✓
(c) *very pretty = AP ✗
(d) *go home = VP ✗

→ AP, VP는 focus position에 올 수 없음 → category 판별 테스트로 활용
Q18빈칸
다음 빈칸에 알맞은 말을 영어로 쓰시오.

*silly and these ideas가 비문인 이유는, Determiner(these)는 Adjective(silly)와 ________ 될 수 없기 때문이다.
정답: coordinated
Determiners cannot be coordinated with Adjectives가 핵심 원칙입니다.
→ *silly and these ideas ✗ / *my and tall son ✗
cf. silly and boring ideas ✓ (Adj + Adj coordination 가능)
이 테스트로 Determiner와 Adjective를 구별할 수 있습니다.
Q19O / X
다음 문장의 문법성을 판단하시오.
He collapsed while the party was in full swing.
정답: ⭕ 정문
while은 Clause complement만 허용합니다.
[the party was in full swing]은 Clause → while의 subcategorization 조건 충족 → 정문
cf. *He collapsed while the party. ✗ (NP complement 불가)
cf. *He collapsed while. ✗ (complement 없음 불가)
Q20객관식
Radford의 입장에 따르면, Adjective와 Adverb를 하나의 범주로 통합하여 'Advective'라고 부를 수 있는 근거로 가장 적절한 것은?
정답: ②
Adjective와 Adverb는 동일한 범주적 특성을 공유합니다.
· 둘 다 very의 수식을 받음 (very happy / very quickly)
· 둘 다 비교급·최상급 형성 가능 (happier / more quickly)
· 서로 coordination 가능 (happy and quickly — predicative 위치에서)
→ 이를 근거로 Radford는 'Advective'라는 단일 범주로 통합 제안
Q21객관식
Radford가 Determiner를 Adjective의 하위 범주로 재분석하는 것을 거부하는 근거로 가장 적절한 것은?
정답: ③
Adjective와 Adverb는 통합 가능한 근거가 있지만, Determiner는 다릅니다.
· *very the book ✗ → very 수식 불가
· *the-er book ✗ → 비교급 불가
· *my and tall son ✗ → Adjective와 coordination 불가
→ Determiner는 Adjective와 범주적 특성이 다름 → 별도 범주 유지
Q22빈칸
다음 빈칸에 영어로 쓰시오.

Radford에 따르면 범주(category)는 원시적(primitive) 단위가 아니라, major/minor ________ feature의 복합체(composite)이다.
정답: categorial
Radford §3.8의 핵심 주장: 범주는 major/minor categorial features의 복합체입니다.
예: N = [+N, -V], V = [-N, +V], A = [+N, +V], P = [-N, -V]
단일 letter symbol(N, V 등)은 편의상 표기이며, 실제로는 feature complex입니다.
Q23O / X
다음 문장의 문법성을 판단하시오.
It was a car that she bought.
정답: ⭕ 정문
Cleft sentence의 focus position에 NP [a car]가 위치 → ✓
→ 정문 (NP는 cleft focus position에 올 수 있음)
이 테스트를 통해 focus 위치의 성분이 NP 또는 PP임을 확인할 수 있습니다.
Q24객관식
Radford가 제시하는 범주 판별 기준과 결과가 모두 옳게 연결된 것은?
정답: ③
Radford §3.6의 핵심 주장: Particle, Conjunction, Preposition은 모두 Preposition으로 통합.
· Particle: complement 없음 (He's been difficult ever since.)
· Conjunction: Clause complement (He's been difficult ever since the party began.)
· Preposition: NP complement (He's been difficult ever since the party.)

④ 오답: Adjective ↔ Adverb는 통합 가능 → 'Advective' 제안 (§3.7)
